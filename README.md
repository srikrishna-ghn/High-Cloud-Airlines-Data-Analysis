# High Cloud Airlines – Data Analysis Project

## Overview
This project analyzes airline operational data to generate insights on passenger trends, route efficiency, and overall performance using Excel, SQL, and Power BI.

## Repository Contents

- `main_data.xlsx ` – Raw dataset 
- `analysis_queries.sql` – 15+ SQL queries for data analysis  
- `dashboard.pbix` – Power BI dashboard for visualization  

## Work Done
- Cleaned and prepared raw dataset using Excel  
- Developed 15+ SQL queries to extract insights and perform analysis  
- Built an interactive dashboard using Power BI 

## Key Insights
- Passenger trends across time periods  
- Route efficiency and performance analysis  
- Carrier-wise operational insights  

## Tools Used
- Excel  
- SQL  
- Power BI
## Outcome
- Transformed raw data into structured insights  
- Built an end-to-end analytics workflow for decision support
